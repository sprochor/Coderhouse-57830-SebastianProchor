Descripción del Proyecto

Este proyecto es una aplicación para administrar nóminas de recursos humanos y la liquidación de sueldos. Tiene varias secciones:

  • Legajos: Puedes ver, crear y editar legajos.
  • Novedades: Se pueden agregar novedades a cada legajo, como ausencias.
  • Liquidación: Se liquida el sueldo mensual, considerando ausencias y cambios de ingreso o egreso.

Mejoras Futuras

Planeo realizar las siguientes mejoras en el proyecto:

  • Separar el legajo según datos básicos, domicilio, centro de costo, cuenta bancaria, obra social, y puesto/sueldo.
  • Cargar todo tipo de ausencias, licencias y novedades como horas extras en el campo de novedades.
  • Detallar más la liquidación y generar recibos de sueldo en formato PDF.
  • Crear una nueva app para editar y parametrizar tablas como países, bancos, obras sociales, puestos, aportes y contribuciones.
